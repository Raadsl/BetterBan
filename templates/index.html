About me

Some time ago I was in a Discord server where a user spammed and acted inappropriatly, after that he left. And the mods couldn't ban him! So I developed this bot to ban users who aren't in your server. It has a list and when a user joins the server it checks the list. If the userID of that user is in the list it bans the user. It even supports ban reason and tempbans!

You can see all the commands when you run the /help command, but here are some of the most important commands:

  • /bban - The main command, Bans a user from your guild, even if they aren't in your server!
  • /set_log_channel - Set the channel to log messages of the bot. Here will all logs from the bot go.
  • /listbban - Gets a list of all the bbanned users.
